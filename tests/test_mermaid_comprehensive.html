Complex (autonumber, alt/loop/par blocks)

sequenceDiagram
    autonumber
    participant C as Client
    participant LB as Load Balancer
    participant S1 as Server 1
    participant S2 as Server 2
    participant DB as Database

    C->>LB: Request
    alt Server 1 available
        LB->>S1: Forward
        activate S1
        S1->>DB: Query
        DB-->>S1: Results
        S1-->>LB: Response
        deactivate S1
    else Server 2 fallback
        LB->>S2: Forward
        activate S2
        S2->>DB: Query
        DB-->>S2: Results
        S2-->>LB: Response
        deactivate S2
    end
    LB-->>C: Response

    loop Every 30s
        C->>LB: Heartbeat
        LB-->>C: ACK
    end

    par Background Tasks
        S1->>DB: Cleanup old sessions
    and
        S2->>DB: Sync replicas
    end

    Note right of C: Connection closed
